# BannerBlaze

Landing page oficial de **BannerBlaze** — Publicidad digital exterior interactiva.

🌐 **Web:** [bannerblaze.com](https://bannerblaze.com)
📍 **Ubicación:** Medellín, Colombia

## Sobre el proyecto

BannerBlaze convierte espacios públicos en experiencias digitales interactivas. Pantallas que capturan la mirada, contenido que se explora desde el celular, y publicidad verificada antes de salir al aire.

## Stack

- HTML5 + CSS3 + JavaScript vanilla
- Tipografías: Archivo Black + Manrope (Google Fonts)
- Sin dependencias externas
- 100% responsive (mobile-first)
- Deploy en Vercel

## Estructura

```
bannerblaze-web/
├── index.html          # Landing principal
├── README.md           # Este archivo
└── .gitignore          # Archivos ignorados por Git
```

## Desarrollo local

Solo abre `index.html` en cualquier navegador. No requiere build ni servidor.

## Deploy

Cualquier push a la rama `main` despliega automáticamente en Vercel.
